# Semana 3: Proyectos Integrales de Análisis de Datos

Desarrollo y aplicación de técnicas de análisis exploratorio en múltiples contextos. Esta semana consolida los conocimientos previos mediante proyectos prácticos con diferentes dominios de datos.

## Descripción General

Se abordan tres proyectos independientes que permiten aplicar el flujo completo de análisis exploratorio: desde carga y limpieza hasta visualización e interpretación de resultados.

## Proyectos

### Proyecto Fintech
Análisis de datos financieros aplicando técnicas de EDA en contexto empresarial:
- Exploración de transacciones y patrones financieros
- Identificación de anomalías
- Visualización de tendencias temporales

### Proyecto Iris
Análisis clásico del dataset Iris (flores):
- Exploración de características biométricas
- Análisis de distribuciones por especie
- Correlaciones entre variables medidas

### Opción 1
Proyecto adicional que varía según necesidades específicas:
- Práctica complementaria en análisis exploratorio
- Aplicación en diferentes contexto

## Contenido General

Cada proyecto incluye:
- Scripts de carga y limpieza de datos
- Análisis exploratorio completo
- Visualizaciones interpretables
- Documentación de hallazgos

## Objetivos de Aprendizaje

- Aplicación práctica de técnicas de EDA en contextos reales
- Limpieza y preparación de datos para análisis
- Selección y evaluación de características relevantes
- Creación de visualizaciones informativas
- Interpretación crítica de patrones en datos
- Documentación de procesos analíticos

## Conceptos Clave

- Flujo de trabajo EDA completo
- Manejo de diferentes tipos de datos
- Análisis bivariado y multivariado
- Visualización exploratorio avanzada
- Generación de hipótesis basada en datos

## Cómo Utilizar Este Módulo

1. Selecciona un proyecto específico
2. Carga y explora el dataset correspondiente
3. Realiza limpieza exploratorio de datos
4. Genera visualizaciones múltiples
5. Documenta hallazgos y patrones identificados
