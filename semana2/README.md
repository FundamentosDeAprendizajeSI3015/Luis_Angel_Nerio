# Semana 2: Análisis Introductorio de Datos

Introducción a los fundamentos del análisis de datos utilizando el dataset Wine (vinos). Esta semana cubre las habilidades esenciales para trabajar con conjuntos de datos, desde la carga inicial hasta la exploración visual.

## Descripción General

Este módulo proporciona una base sólida en manipulación y análisis exploratorio de datos. Utiliza el dataset Wine, un conjunto de datos de referencia que contiene información sobre propiedades químicas de diferentes vinos.

## Contenido

- **Dataset_wine/**: Carpeta principal con datasets, scripts y primeros análisis
  - Archivos de datos en formato CSV o similar
  - Scripts de carga y manipulación de datos
  - Visualizaciones inicialesArchivos de configuración y utilidades

## Archivos Principales

- Scripts de carga de datos: Demuestran cómo importar y estructurar datos en memoria
- Scripts de exploración: Análisis estadísticos básicos (media, mediana, desviación estándar)
- Visualizaciones: Gráficos iniciales para entender patrones y distribuciones

## Objetivos de Aprendizaje

- Carga y manipulación básica de datos con pandas/numpy
- Exploración exploratoria de datasets (EDA introductorio)
- Fundamentos de visualización de datos con matplotlib/seaborn
- Cálculo de estadísticas descriptivas
- Identificación de características y tipos de datos

## Conceptos Clave

- Estructuras de datos (DataFrames, arrays)
- Operaciones básicas (filtrado, selección de columnas)
- Visualización de una variable (histogramas, boxplots)
- Análisis univariado

## Cómo Utilizar Este Módulo

1. Explora la estructura del dataset Wine
2. Carga los datos en Python
3. Calcula estadísticas descriptivas
4. Genera visualizaciones básicas para entender los datos
